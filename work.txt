Создать аккаунт на GitHub
Перейти на репозитория преподавателя (fork)
Скопировать ссылку на репозиторий
Перейти в VScode и с помощь команды cd Desktop перейти на рабочий стол
Вызовите команду git clone+ссылка сохранить папку преподавателя
 Вызвать команду cd+название папки и открыть папку
Создать новую ветку и перейти в нее с помощью команды git checkout -b+name of branch
Создать новый файл в ветке с помощью команды touch+name_of_file+расширение файла(txt,md и т.д)
Создать коммит и добавить в файл
Отправиь запрос на слияние ветки на аккаунт преподавателя с помощью команды git push -u origin+name_of_branch
